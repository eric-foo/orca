# Synthesis

Store reasoning outputs that evaluate gathered consulting-corpus evidence.

Synthesis artifacts may score candidates, classify Tier A/B/C/Reject, compare firms and source types, recommend first backtest candidates, and state limitations.

Every synthesis artifact should cite the lane outputs or external URLs it relies on and keep these boundaries explicit:

- consulting benchmark value;
- actual backtestability;
- external outcome evidence;
- anonymous marketing claims.
