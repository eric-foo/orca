# Backtestability

Store external evidence hooks and pre-backtest notes for consulting-corpus candidates.

Use this folder to record public sources that may later support a real Orca backtest, such as client press releases, annual reports, government reports, investor materials, earnings transcripts, reputable news coverage, public procurement pages, or official implementation pages.

This folder does not execute backtests, lock cases, create source systems, or prove Orca performance.
