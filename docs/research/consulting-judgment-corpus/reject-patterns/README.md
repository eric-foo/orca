# Reject Patterns

Store maps of consulting-source patterns that look promising but fail the corpus criteria.

Examples:

- thought leadership without a concrete case;
- service or capability pages;
- AI transformation essays with no named case;
- metric-only client impact pages;
- anonymized cases with no external reconstruction path;
- expert interviews without situation, diagnosis, recommendation, action, and result.

Record the search query, source URL, missing fields, and short evidence excerpts where useful.
