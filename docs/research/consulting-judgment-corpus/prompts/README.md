# Prompts

Store prompts used to launch consulting-judgment-corpus research lanes.

Recommended naming:

- `YYYY-MM-DD_o3_lane_01_structured_case_repositories_prompt_v0.md`
- `YYYY-MM-DD_o3_lane_02_strategy_firm_judgment_cases_prompt_v0.md`
- `YYYY-MM-DD_o3_lane_03_decision_family_search_prompt_v0.md`
- `YYYY-MM-DD_o3_lane_04_backtestability_evidence_screen_prompt_v0.md`
- `YYYY-MM-DD_o3_lane_05_reject_pattern_mapping_prompt_v0.md`
- `YYYY-MM-DD_gpt55_synthesis_prompt_v0.md`

Evidence-only prompts should state that the model must gather evidence only and must not score, rank, synthesize, recommend, or infer strategic value.
