# Orca Docs

This folder holds Orca-owned docs-first artifacts. It does not contain software implementation.

- `decisions/`: decision records.
- `product/`: product contracts, core-spine notes, satellite notes, evidence standards, source maps, and decision-memo shape drafts.
- `prompts/`: prompt artifacts.
- `review-inputs/`: artifacts prepared for review.
- `review-outputs/`: reviewer reports.
- `workflows/`: workflow records and bootstrap notes.
- `migration/`: import and migration planning records.
- `hygiene/`: triage queues and cleanup notes.
- `_inbox/`: non-authoritative scratch for temporary prompts, notes, imports, and untriaged material.

See `STRUCTURE.md` before adding new docs folders or promoting material out of `_inbox/`.
